## **Проекты**

### **Fit quest**

*(весна 2025)*

**Стек:** Scala | Cats Effect | Http4s | Doobie | Postgresql | Docker

Небольшой сервис мониторинга здоровья, интегрированный с Google Health. Это был мой первый опыт с typelevel бэкендом. Его код есть в открытом доступе:
[github.com/andreyegor/fit-quest](https://github.com/andreyegor/fit-quest).

* Реализовал REST API для работы с пользовательскими данными и тренировками.
* Спроектировал и реализовал систему аутентификации на access/refresh токенах с использованием http4s middleware.
* Организовал интеграцию с Google Health Connect: ручная синхронизация и фоновое обновление данных по расписанию.
* Использовал функциональный подход (tagless final) для разделения бизнес-логики и эффектов.
* Реализовал работу с PostgreSQL через Doobie (включая транзакции и маппинг доменных моделей).
* Развернул это всё на сервере в Docker через Nginx

Также мной было написано небольшое приложение для интеграции с Google Health Connect, можно залогиниться и синхронизировать вручную, автосинхронизация проходит два раза в сутки.
