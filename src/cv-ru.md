### Бэкенд разработчик для Томского Политехнического Университета

*(Июль 2025 - Сентябрь 2025)*

**Стек:** Scala | Cats Effect | Http4s | Doobie | Postgresql | Docker

* Разработал mvp бэкенд ed-tech сервиса тпу по поиску студенческих проектов на Scala Typelevel.
* Использовал Oauth *(CRUD для проектов, пользователей и заявок, логика модерации проектов, логика заявок на роли с проверкой доступности)*
* Занимался поддержкой сервиса некоторе время после запуска.
* Настроил простой деплой используя docker, nginx и github actions. Развернул beszel для минимального контроля работы сервиса.
* В первый сезон работы больше 100 студентов подали заявку на проект через сервис.
* **Сейчас мы продолжаем развивать сервис, но уже как pet проект.**

### **Fit quest**

*(весна 2025)*

**Стек:** Scala | Cats Effect | Http4s | Doobie | Postgresql | Docker

Небольшой сервис мониторинга здоровья, интегрированный с Google Health. Это был мой первый опыт с typelevel бэкендом и его код есть в открытом доступе:
[github.com/andreyegor/fit-quest](https://github.com/andreyegor/fit-quest) (но сейчас я пишу получше).

Бэкэнд написан полностью в функциональном стиле с использованием tagless final подхода. Без использования готовых решений написана система авторизации с аутентификацией на основе access/refresh токенов и http4s middleware. Может хранить и отдавать данные о тренировках по запросу.

Также мной было написано небольшое приложение для интеграции с Google Health Connect, можно залогиниться и синхронизировать вручную, автосинхронизация проходит два раза в сутки.
